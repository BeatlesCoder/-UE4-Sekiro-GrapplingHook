# -UE4-Sekiro-GrapplingHook | MbM 

Retro engineering of the grappling hook mechanic from Sekiro: Shadows Die Twice in UE4.

## Prerequisites

-  UE4 4.25 or higher


## Instructions

- WASD/Left Stick: Movement.
- Space/A: Jump.
- Mouse/Right Stick: Camera Control.
- F/Left Trigger: Grappling Hook.
